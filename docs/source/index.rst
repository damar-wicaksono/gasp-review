Gaussian Process Metamodeling: Review and Example of Applications
=================================================================

This is a documentation of a project to review the gaussian process metamodeling
using an ``R`` package called ``DiceKriging``.
The review is based on several test cases ranging from simple analytical function
to more complex one using code.
The final document is released as a technical report with the following metadata.
